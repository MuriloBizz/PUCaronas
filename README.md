# PUCaronas
<img width="1600" height="653" alt="image" src="https://github.com/user-attachments/assets/3b3fab15-55d7-4cf9-9017-6b544c50c41f" />

Uma aplicação web que conecta estudantes da Pontifícia Universidade Católica do Paraná para compartilhamento de transportes, reduzindo custos, tempo de deslocamento e aumentando segurança através de um sistema de avaliação e verificação de usuários.
